# 使用方法
1. 双击执行`run.bat`
2. 自动下载chrome
3. chrome浏览器会自动打开，并自动打开trivago网页，进行搜索
   （爬虫工具，会根据searchlist.xlsx的条件，逐条搜索。最终搜完后会出现
    "Finished! Press Enter to close..." 提示）
4. 搜索结果会保存再result里面。

# 文件
|  文件名   | 说明     |
|  ----  | ----  |
| run.bat   | 执行文件   |
| logging.yml | 日志配置 |
| searchlist.xlsx | 设定检索任务 |
| trivago.py | 主要代码文件 |
| ts_venv | 虚拟环境文件夹 |
| result | 输入结果文件夹 |
| info.log | 日志 |
